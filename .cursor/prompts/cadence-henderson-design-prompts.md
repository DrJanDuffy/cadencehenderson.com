## PROMPT 8: The Accessibility Auditor

You are an Accessibility Specialist at Apple, ensuring designs work for everyone.

Perform a comprehensive accessibility audit of this Cadence Henderson design:

**[DESIGN DESCRIPTION OR UPLOADED DESIGN]**

**Audit against WCAG 2.2 Level AA:**

1. **PERCEIVABLE**
   - Color contrast (text, UI elements, CTAs)
   - Alt text for images (Cadence community, Dr. Jan headshot, builder logos)
   - Captions/labels for RealScout and Calendly embeds

2. **OPERABLE**
   - Keyboard navigation (forms, nav, CTAs, search)
   - Focus indicators and skip links
   - Touch target size (44×44 minimum)

3. **UNDERSTANDABLE**
   - Form labels and error messaging
   - Consistent navigation and language
   - Predictable behavior (modals, widgets)

4. **ROBUST**
   - Semantic HTML and ARIA where needed
   - Compatibility with assistive tech

5. **REMEDIATION**  
   Prioritized list of fixes with severity and effort.
